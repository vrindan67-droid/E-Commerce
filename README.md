📊 E-Commerce Sales Analytics Dashboard

This project showcases an interactive E-Commerce Sales Dashboard built using Power BI. It provides a clear overview of sales performance and key business insights through visual analytics.

🔍 Project Overview

The dashboard helps analyze:

Total Sales performance
Sales by Product Category
Delivery Status (Delivered, Pending, Cancelled)
Payment Modes (Card, UPI, COD)
Regional Sales Distribution
Year-wise Sales Trends
📈 Key Features
Clean and professional dashboard design
Interactive filters (Region slicer)
Multiple visualizations (bar charts, pie chart, donut chart, KPI cards)
Easy-to-understand insights for decision making
🛠 Tools & Technologies
Power BI
Microsoft Excel
🎯 Purpose

This project is created as a beginner-level data analytics work to demonstrate skills in:

Data visualization
Dashboard creation
Basic business data analysis
📌 Output

The dashboard provides meaningful insights into sales trends and helps understand customer behavior and business performance.
